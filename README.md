# Oxygen | v1.6
* By LiteClubDevelopment, Traduciendo
* Available for download on [BuiltByBit](https://builtbybit.com/resources/oxygen-modern-bungee-core.54077/) and [GitHub](https://github.com/HCFAlerts/Oxygen/releases)
---

### Features
- Optimized  
- Fully Customizable (Via YML)  
- MOTD:  
  - With Countdown Fully Customizable  
  - Slots Customizable  
  - Ping Text Customizable (With Maintenance Mode)  
- Media Commands:  
  - Discord  
  - Teamspeak  
  - Website  
  - Twitter  
  - Store  
- Help Command  
- Find Command  
- Hub Command  
- Server Command  
- Maintenance Mode  
- Kick System (With Auto Hub Detection)  
- And big loot of more features...
