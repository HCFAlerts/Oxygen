## Features
* Maintenance Mode: /maintenance <on|off|list|add|remove> <name>
* Media Commands: /<discord|teamspeak|store|website|twitter>
* Hub Command: /<hub|lobby>
* Misc Commands: /<server|find>
* Kick System: Send player to the Hub
